How to install (.RBXL)?

  • Download the RBXL file from teamgeekyt.github.io/ezscripty. Other websites are not official!
  • Open it with Roblox Studio.
  • Done! You now have EZScripty!

How to install (.LUAU)?

  • Download the LUA file from teamgeekyt.github.io/ezscripty. Other websites are not official!
  • Open it with any text software.
  • Copy the entire code.
  • Create an ModuleScript in ServerScriptService named "EZScripty"
  • Paste the code in it
  • Done! You now have EZScripty!
